# Laravel

PHP로 구현된 Laravel 어플리케이션 템플릿입니다.

## 🖇️ 준비 및 확인사항

### 지원 PHP 버전
- 7.3, 7.4, 8.0, 8.1, 8.2 (배포 시 자동 탐지)

### 필요 파일
- 빌드 시 어플리케이션을 빌드하기 위해서 `Procfile`이 필요합니다.
